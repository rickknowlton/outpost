# Lancaster Pet Grooming - Landing Page

A simple Next.js landing page for Lancaster Pet Grooming featuring the business logo and an embedded MoeGo appointment scheduler.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```

## Adding Your Logo

1. Place your logo image file (PNG, SVG, or JPG) in the `public` folder
2. Update `app/page.tsx` to uncomment the Image component and comment out the placeholder
3. Update the `src` path to match your logo filename

## Deployment

This project is configured for deployment on Vercel. Simply push to GitHub and connect your repository to Vercel for automatic deployments.

## Features

- Responsive design that works on all devices
- Embedded MoeGo appointment scheduler
- Clean, modern UI with gradient background
- Ready for Vercel deployment
